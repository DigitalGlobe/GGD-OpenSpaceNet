Error * dg::deepcore::imagery::MapServiceClient::onHttpError ( const Error &  e ) const
protectedvirtualnoexcept

HTTP Error callback method. Note that this method will only be invoked for exceptions of type HttpError. The implementing class can override this function to interpret and optionally transform the input HttpError. If the Error does not need to be transformed, nullptr should be returned. Caller takes ownership of the pointer.

Parameters
[in] e The input HttpError.
Returns
A new instance of Error, or nullptr if Error was not transformed.
