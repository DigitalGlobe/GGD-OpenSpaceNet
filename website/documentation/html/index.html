DeepCore

Introduction

DeepCore is a library framework that will allow application developers to leverage deep machine learning while encapsulating some of the complexity. This building block approach will allow developers to use common functionality from the DeepCore framework, putting the business logic in the application where it belongs.

Dependencies

DeepCore has dependencies on several libraries, such as:

  • OpenCV for image manipulation and loading images for classification
  • Caffe for image classification
  • Boost for general toolkit things
  • GDAL/OGR for geospatial exports
  • CURL / Curlpp for web calls and downloads
  • libjson-rpc-cpp for json rpc calls Prior to building, run the osn_install.sh script to install all of the dependencies

Building

mkdir build
cd build
cmake .. -DCMAKE_CXX_FLAGS="-I /opt/DeepCore/include -L/opt/DeepCore/lib" -DCMAKE_PREFIX_PATH=/opt/DeepCore
make
sudo make install

This will install the framework libs into /usr/lib/deepcore, for later use with the apps
